Datasets en Python

Un dataset es la materia prima del análisis de datos. Es una colección estructurada de información organizada en filas (observaciones) y columnas (variables). En Python, los DataFrames de pandas son la herramienta estándar para manipularlos.

Tipos de Variables en un Dataset

Los datos pueden clasificarse en:

  • Cuantitativos Continuos: Números con infinitos valores posibles (ej: temperatura, peso).
  • Cuantitativos Discretos: Números enteros (ej: número de hijos, artículos comprados).
  • Cualitativos Nominales: Categorías sin orden (ej: género, ciudad).
  • Cualitativos Ordinales: Categorías con jerarquía (ej: nivel educativo: "básico", "medio", "avanzado").

Importante: El tipo de variable determina qué análisis estadísticos puedes aplicar.

Ejemplo: Creación de DataFrame 
import pandas as pd

# Crear DataFrame desde diccionario
datos = pd.DataFrame({
    'Nombre': ['Juan', 'Ana', 'Pedro', 'Laura', 'María'],
    'Edad': [23, 45, 31, 35, 22],          # Variable cuantitativa
    'Ciudad': ['Bogotá', 'Medellín', 'Bogotá', 'Cali', 'Medellín'],  # Cualitativa
    'Ingresos': [2000, 3000, 4000, 5000, 6000]
})

# Replicar datos para ejemplo realista
datos = pd.concat([datos] * 1000, ignore_index=True)

¿Por qué usar DataFrames?

Los DataFrames de pandas ofrecen:

  • Manipulación eficiente: Filtrado, agrupación y transformación de grandes volúmenes de datos.
  • Integración: Compatibilidad con bibliotecas como NumPy, Matplotlib y Scikit-learn.
  • Limpieza de datos: Funciones para manejar valores faltantes (NaN) y outliers.

Estadística Descriptiva

La estadística descriptiva resume y describe las características principales de un dataset. Se divide en:

Medidas de Tendencia Central

Indican el valor "típico" o central de los datos:

  • Media: Promedio aritmético. Útil para datos simétricos sin outliers.
  • Mediana: Valor que divide el dataset en dos partes iguales. Ideal para datos sesgados.
  • Moda: Valor más frecuente. Única medida aplicable a datos categóricos.

Ejemplo práctico: En salarios, la mediana es más representativa que la media si hay pocos salarios muy altos.

# Cálculo con NumPy
        import numpy as np
        datos = [5, 8, 6, 9, 8, 5, 10, 7, 6, 6, 100]  # 100 es outlier

        media = np.mean(datos)     # 15.0 (distorsionada)
        mediana = np.median(datos) # 7.0 (representativa)

Medidas de Dispersión

Miden cuánto varían los datos:

  • Rango: Diferencia entre el valor máximo y mínimo. Sensible a outliers.
  • Varianza: Promedio de las diferencias al cuadrado respecto a la media.
  • Desviación Estándar: Raíz cuadrada de la varianza. Expresada en las mismas unidades que los datos.

Regla empírica: En distribuciones normales, el 68% de los datos está dentro de ±1 desviación estándar de la media.

rango = np.ptp(datos)  # 95
        varianza = np.var(datos, ddof=1)  # 736.4
        desviacion = np.std(datos, ddof=1)  # 27.14

¿Outliers? ¡Cuidado!

Los outliers son valores anómalos que distorsionan los análisis. Para detectarlos:

  • Método IQR: Cualquier valor fuera de Q1 - 1.5*IQR o Q3 + 1.5*IQR.
  • Gráficos: Boxplots y scatter plots son ideales para visualizarlos.
Q1 = datos['Ingresos'].quantile(0.25)
        Q3 = datos['Ingresos'].quantile(0.75)
        IQR = Q3 - Q1
        outliers = datos[(datos['Ingresos'] < (Q1 - 1.5*IQR)) | (datos['Ingresos'] > (Q3 + 1.5*IQR))]

Probabilidad con Python

La probabilidad cuantifica la incertidumbre de eventos futuros. En ciencia de datos se usa para:

  • Modelar comportamientos aleatorios (ej: tráfico web, fallos en equipos).
  • Diseñar algoritmos (ej: A/B testing, redes neuronales).
  • Calcular riesgos (ej: préstamos bancarios, diagnósticos médicos).

Conceptos Clave

  • Espacio Muestral: Conjunto de todos los resultados posibles (ej: {1,2,3,4,5,6} en un dado).
  • Evento: Subconjunto del espacio muestral (ej: "sacar número par").
  • Probabilidad Clásica: P(A) = Casos favorables / Casos posibles.
  • Ley de los Grandes Números: Al repetir un experimento muchas veces, la frecuencia relativa converge a la probabilidad teórica.
Simulación de Lanzamiento de Dados 
import numpy as np

# Simular 1 millón de lanzamientos
dados = np.random.randint(1, 7, 1000000)

# Calcular probabilidades
for i in range(1, 7):
    prob = (dados == i).mean()
    print(f"Probabilidad de {i}: {prob:.2%}")
